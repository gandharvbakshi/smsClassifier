# Google Play Store Assets Checklist

Use this checklist to ensure you have all required assets before submitting to Google Play.

## Required Assets

### ✅ App Icon
- [ ] Created 512x512 pixel icon
- [ ] Format: PNG or JPEG
- [ ] File size under 1 MB
- [ ] No transparency (solid background)
- [ ] Tested at small sizes (48x48px)
- [ ] Saved as: `app-icon.png`

### ✅ Feature Graphic
- [ ] Created 1024x500 pixel graphic
- [ ] Format: PNG or JPEG
- [ ] Includes app name: "SMS Classifier"
- [ ] Highlights key features
- [ ] Text is readable
- [ ] Saved as: `feature-graphic.png`

### ✅ Screenshots
- [ ] At least 2 screenshots taken
- [ ] All screenshots are 16:9 or 9:16 aspect ratio
- [ ] All show actual app functionality
- [ ] Text is readable
- [ ] No sensitive information visible
- [ ] Saved in `screenshots/` folder

## Store Listing Text

### ✅ App Name
- [ ] App name: "SMS Classifier" (or your choice)
- [ ] Under 30 characters
- [ ] No trademark violations

### ✅ Short Description
- [ ] Under 80 characters
- [ ] Example: "Smart SMS classifier for OTP detection and phishing protection"

### ✅ Full Description
- [ ] Under 4000 characters
- [ ] Includes key features
- [ ] Mentions OTP detection
- [ ] Mentions phishing protection
- [ ] Includes feature list

## Optional Assets

### Promotional Video
- [ ] YouTube video created (optional)
- [ ] 30 seconds to 2 minutes
- [ ] Demonstrates app functionality

### Tablet Screenshots
- [ ] Only if supporting tablets
- [ ] Same requirements as phone screenshots

## Upload to Google Play Console

### Store Listing Section
- [ ] App name entered
- [ ] Short description entered
- [ ] Full description entered
- [ ] App icon uploaded
- [ ] Feature graphic uploaded
- [ ] At least 2 screenshots uploaded

### App Content Section
- [ ] Privacy Policy URL added
- [ ] Data Safety form completed
- [ ] Content Rating completed

### Release Section
- [ ] AAB file uploaded (version code 2+)
- [ ] Release notes written

## Final Checklist

- [ ] All required assets created
- [ ] All assets meet size/format requirements
- [ ] Store listing text completed
- [ ] Privacy policy URL working
- [ ] Data Safety form completed
- [ ] Content Rating completed
- [ ] AAB uploaded
- [ ] Ready to submit for review

## Quick Reference

| Asset | Size | Format | Status |
|-------|------|--------|--------|
| App Icon | 512x512 | PNG/JPEG | ⬜ |
| Feature Graphic | 1024x500 | PNG/JPEG | ⬜ |
| Screenshots | 16:9 or 9:16 | PNG/JPEG | ⬜ (min 2) |

## Notes

- All images should be optimized for web (compressed but still high quality)
- Test all assets on different devices/screen sizes
- Ensure all text is readable
- Follow Google Play design guidelines
- Review all assets before final submission
