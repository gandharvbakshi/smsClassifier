# Creative Strategy

## Batch Direction

UI-led Google App Campaign assets focused on product proof instead of text-only posters. Every image variation uses a visible SMS Classifier UI screen, with no baked install/download CTA.

## Concept Families

1. OTP inbox: find OTPs faster.
2. Suspicious review: review risky-looking texts before acting.
3. Sorted inbox: keep SMS categories organized.
4. Correction logs: fix wrong labels and improve review workflows.
5. Pro settings: explain trial/Pro cloud checks without overclaiming.
6. Data controls: show privacy and cloud review controls honestly.

## Production Rules

- Use app UI as the hero visual.
- Keep on-image text light for Google App Campaign compatibility.
- Put CTA text in platform fields, not burned into image art.
- Avoid absolute safety, guaranteed protection, or bare-free wording.
