# Product Context

## Product

SMS Classifier is an Android SMS inbox and classification helper. It helps people find OTPs, review suspicious-looking messages, keep service/general messages organized, and correct labels when classification is wrong.

## Audience

Android users in India who receive OTPs, delivery/service notifications, and risky-looking SMS messages and want a clearer review workflow.

## Evidence Used

- Store listing text: `android_sms_classifier/store_assets/store-listing-text.md`.
- Existing icon export bundle: `ads/icon-assets/`.
- Existing Play Store screenshots reviewed for UI structure but not reused directly because several contain demo phone-like numbers, brand names, codes, and absolute `Safe` labels.
- Sanitized reusable UI proofs: `ads/_shared/screenshots/sanitized_demo/`.

## Claims Allowed

- Helps find OTPs faster.
- Helps sort OTP, general, suspicious, and review-needed messages.
- Helps review suspicious-looking texts before acting.
- Trial/Pro can enable cloud classification features.
- Users can export data or delete local app data from Settings.

## Claims To Avoid

- Do not claim messages are always safe.
- Do not claim all scams are blocked.
- Do not claim messages never leave the device when cloud checks are active.
- Do not use bare free CTAs such as `Install free` or `Download free`.

## Offer

Use trial-qualified CTAs only: `Free for 14 days`, `Start free trial`, `Try it free`, or neutral store CTAs.
