# Reusable SMS Classifier Ad Inputs

Use this folder as the source shelf for future ad runs.

- `brand_kit.yaml`: reusable brand colors, logo references, tone, and forbidden claims.
- `monetization.yaml`: current pricing/trial facts used by CTA and pricing linters.
- `product_profile.md`: concise reusable app facts and claim guardrails.
- `icons/`: canonical ad-ready app icon copies.
- `screenshots/sanitized_demo/`: reusable sanitized UI proof screens for draft ads.

The sanitized demo screens are ad-safe mock/demo screens based on the app UI pattern. Replace them with fresh live app screenshots before a final paid launch when current UI accuracy matters.
