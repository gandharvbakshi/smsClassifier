# SMS Classifier Product Profile

## Reusable App Facts

- Product: SMS Classifier.
- Category: Android SMS inbox and classification helper.
- Core promise: make OTPs, service messages, suspicious-looking texts, and review-needed messages easier to find and review.
- Required setup: full SMS features require becoming the default SMS app.
- Free/trial/Pro: basic local OTP detection is available; a 14-day Pro trial and active Pro subscription can enable cloud classification features.
- Pricing fact: Pro is an annual Google Play subscription listed as INR 199/year in India.
- Privacy fact: messages are stored locally for inbox features; cloud classification can send message text and sender info over HTTPS when trial/Pro cloud checks are active.

## Claim Guardrails

Use `helps`, `highlights`, `easier to review`, and `decision aid`. Avoid absolute claims such as `safe`, `100% safe`, `blocks all scams`, `never miss`, or `messages never leave device`.

## Full Description

SMS Classifier is an SMS-first inbox for people who want OTPs, service messages, and risky-looking texts to be easier to review.

It can be set as your default SMS app so it can receive, read, send, and organize messages on your phone.

What it helps with

- OTP inbox: identifies one-time passwords and makes them easier to find.
- Message labels: helps separate OTP, general, suspicious, and review-needed messages.
- OTP details: shows intent and sensitivity hints when cloud checks are available.
- Scam awareness: highlights phishing signals and risky links when classification data supports it.
- One-tap actions: copy OTPs, search messages, and manage conversations.
- Data controls: export your data or delete local app data from Settings.

Free, trial, and Pro

SMS Classifier includes local OTP detection for basic use. You can choose to start a 14-day Pro trial from setup or the paywall. During the trial, and while your Pro subscription is active, the app can use cloud classification to explain what codes are for, show scam warnings, and surface 'Do not share' alerts. Pro is an annual Google Play subscription priced at ₹199/year in India and renews yearly unless you cancel in Play Store.

Privacy notes

SMS Classifier stores messages in the app's local database so the inbox can work on your device. Cloud classification is used when available through your trial or Pro access and sends message text and sender information over HTTPS to the app's backend for classification. Optional feedback uploads send redacted examples to help improve accuracy. Usage analytics and crash report
