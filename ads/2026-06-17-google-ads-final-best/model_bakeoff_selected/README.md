# Selected Model Bakeoff References

Primary final assets are the controlled UI-led PNG/MP4 files in `assets/` and `exports/`.

These files preserve live model experiments for comparison only. Recraft and Ideogram were the strongest image-background candidates, but deterministic product-proof layouts were selected for shipping. Seedance and Kling produced human scenes with text artifacts or uncontrolled phone text; Veo produced the best human scene but was letterboxed.
